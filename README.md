## Descripción del proyecto

Alerta Cusco es un prototipo de aplicación móvil desarrollado como parte de una investigación orientada a la prevención del delito en ciudadanos de la ciudad de Cusco.

La aplicación permite a los usuarios registrar incidentes ciudadanos, enviar reportes con ubicación GPS, consultar alertas cercanas, visualizar zonas de riesgo, acceder a un mapa delictivo y utilizar un módulo SOS. Además, cuenta con un panel administrador para revisar reportes y visualizar resultados de encuestas enviadas por los usuarios.

El sistema fue desarrollado con el objetivo de apoyar la seguridad ciudadana mediante el uso de tecnologías móviles, geolocalización, almacenamiento en la nube e inteligencia artificial simulada para la clasificación de riesgo.
